Get Started in 30 Seconds

Install and run your first bioinformatics analysis

1

Download

curl -L https://github.com/andyzhuang/openlife/releases/latest/download/openlife -o openlife chmod +x openlife
2

Initialize

openlife onboard
3

Run Analysis

openlife bio query "What drugs should I avoid based on my CYP2D6 gene?"
